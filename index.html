Outcomes: Reduced simulated energy usage by ~12% in test scenarios; dashboard for visualizing consumption.

Challenges & Solutions: Network latency for sensors — solved by local buffering and periodic batch upload; power calibration required manual tuning.

Overview: A tamper-evident ledger for storing grade records and certificates using a permissioned blockchain model.

Methodologies: Hyperledger Fabric (prototype), REST API layer, PostgreSQL for indexing, simple cryptographic signing.

Outcomes: Prototype demonstrated secure record insertion & verification; sample web UI for verification of transcripts.

Challenges: Privacy and access control — addressed via permissions and off-chain encrypted storage.
